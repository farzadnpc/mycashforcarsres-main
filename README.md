Victoria Cash for Cars Website

A modern one-page marketing site for Victoria Cash for Cars, showcasing instant cash offers, free towing and coverage across Melbourne and regional Victoria. The layout introduces the brand with a bold navy and teal palette, a conversion focused quote form, and expandable lists for the makes and council areas the business services.

## Project structure

```
victoriacashforcars/
├── index.html         # Main landing page
├── styles/
│   └── styles.css     # Global styles and responsive layout
├── scripts/
│   ├── script.js      # EmailJS form handling and UX helpers
│   ├── brands.js      # Dynamically renders the brands accordion
│   └── lga.js         # Populates serviced council and suburb accordions
└── images/            # Brand mark and icon assets
```

## Key features

- Hero section with trust metrics, sticky CTA buttons and a fully validated quote form.
- Service highlights, process timeline and assurance copy inspired by leading cash-for-cars sites.
- Accordions for vehicle brands and Victorian council areas powered by JavaScript datasets.
- Responsive, accessible design with high-contrast typography and large tap targets for mobile visitors.

## Getting started

1. Open `index.html` in a web browser to preview the site.
2. Adjust colours, copy or imagery inside `styles/styles.css` and `index.html` as needed.
3. Configure EmailJS credentials in `scripts/script.js` if you plan to use live form submissions.
4. Deploy the static files to your preferred hosting platform.
